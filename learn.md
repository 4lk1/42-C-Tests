# 2-Day Plan to Learn All Exercises

This plan is intense but realistic if you stay focused and code continuously.
Target each exercise with this cycle:
1. Read subject and expected behavior (2-4 min)
2. Write solution from memory (10-25 min)
3. Compile and test with custom cases (5-10 min)
4. Fix edge cases and cleanup (5 min)

Use this compile command template:

```bash
cc -Wall -Wextra -Werror <exercise>.c -o <exercise>
```

If an exercise needs multiple files (for example list exercises), compile with needed files together.
